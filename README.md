# alfred-timeleft

Alfred workflow that shows how much of the current work week, month, and year has elapsed and how much remains. Designed as a barometer for revenue/selling days — weekends and the year-end shutdown (after the third Friday of December) are excluded.

## Install

Download `timeleft.alfredworkflow` from the [latest release](https://github.com/jaacob/alfred-timeleft/releases) and double-click to install.

## Usage

Type `timeleft` in Alfred. Press Enter on any row to copy it, or pick "Copy all" to copy all three.
